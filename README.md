# Análise de Vendas com Python

Projeto de análise de dados desenvolvido em Python para consolidar e analisar múltiplos arquivos CSV de vendas.

## Objetivo
Identificar produtos mais vendidos, maior faturamento e desempenho por loja/cidade.

## Tecnologias
- Python  
- Pandas  
- Plotly  
- Google Colab  

## Descrição
- Leitura automática de arquivos CSV com `os`
- Unificação dos dados em um único DataFrame
- Criação da métrica de faturamento
- Agrupamentos por produto e por loja
- Visualização dos resultados com gráfico de barras
